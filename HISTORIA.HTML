HISTORIA DE HTML


Origen

HTML, es un lenguaje utilizado en programación, el cual permite estructurar, establecer ciertas relaciones, hipervinculos que se pueden integrar en el desde otras paginas diseñadas. Para conocer el origen de HTML es importante remontaros a la decada de los 90, cuando

el físico Tim Berners Lee, investigador de la Organización Europea para la Investigación Nuclear (CERN), realizó una publicación titulada «HTML Tags», conocida como el origen de HTML. Además, unos años después, fundó la W3C, que es uno de los principales agentes en la historia de HTML.

Siendo unos de los principales objetivos la agilización en el intercambio de información en el trabajo. En un principio, su finalidad era que los científicos de diferentes universidades del mundo pudieran compartir información entre sí de manera sencilla. Luego de que su proyecto estuviera listo decidió presentarse a una convocatoria de un sistema de hipertexto para internet, en el cual su propuesta fue la ganadora de la respectiva convocatoria. En el año 1991 se hizo publico el primer documento de HTML. Sin embargo, en 1993 se definieron las etiquetas para tablas, imágenes y formularios, lo que lo llevo a volverse un estandar oficial. A finales de este mismo año se empezó a trabajar en la versión 2. Siendo esta publicada en 1995 como el estándar oficial de HTML.


Versiones

Las versiones de este lenguaje fueron evolucionando de una manera muy rapida, en 1996 apareció en manera de borrador la versión3.0, generando nuevas capacidades y facilidades para mostrar figuras o elementos matemáticos complejos.

El HTML 3.1 nunca fue propuesto de manera oficial, por lo que el siguiente estándar, después de la expiración del anterior, fue el HTML 3.2.

En 1997 por recomendación de W3C se publicó la version 4.0, la cual implementaba hojas de estilos CSS, la mejora de la accesibilidad de las páginas diseñadas, tablas complejas, la posibilidad de incluir pequeños programas o scripts y mejoras en los formularios. En 1999 aparece la última versión oficial de HTML: HTML 4.01, una revisión de la propuesta anterior en la que no se incluyen novedades importantes. En el año 2008 se genero el borrador de la versión HTML5, no obstante este no tuvo mucha relevancia, así pues en el año 2014 las WHATWG se centraron en la ultima versión estandar de HTML, siendo esta la versión 5.2, de hay para adelante se han generado algunas modificaciones pero ninguna tan relevante, que genere un gran impacto en el manejo o estructura del lenguaje de marcado de hipertexto.


Actualidad

HTML es el principal lenguaje de marcado de Internet. Cada página HTML tiene una serie de elementos que crean la estructura del contenido de una página web o aplicación. HTML es un lenguaje apto para utilizar principalmente para páginas web estáticas. HTML funciona mejor junto con CSS para el estilo y JavaScript para la funcionalidad. En cuanto a la versión más reciente de HTML, no olvides que se trata de HTML 5.3, que se emitió en el año 2021, el cual incluye caracteristicas como sporte multimedia (audio y video), gráficos (Canvas), y APIs avanzadas que facilitan el desarrollo web, pero sin duda experimentaremos más y más actualizaciones a lo largo de los años. Sobre todo porque tanto la W3C como la WHATWG siguen realizando trabajos para establecer las recomendaciones y cambios pertinentes para ir generando las nuevas actualizaciones que requiera el lenguaje, para que cada vez sean mas las herramientas y posibilidades de diseño que ofresca el lenguaje de marcado de hipertexto HTML.
